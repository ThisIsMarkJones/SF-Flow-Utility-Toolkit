# SF Flow Utility Toolkit

Utilities for Salesforce Flow Builder and related Salesforce automation pages.

A browser extension designed to make working with Salesforce Flows faster, easier, and more consistent through focused admin, builder, and documentation enhancements.

## Quick links

[What's New](whats-new.html) [Getting Started](getting-started.html) [Features Overview](features/) [Report a Bug](report-a-bug.html) [Feature Requests](feature-requests.html) [Privacy Policy](privacy-policy.html)

## Featured capabilities

### [Flow Health Check](features/flow-health-check.html)

Analyse a Flow and identify maintainability, reliability, performance, and portability concerns.

### [Scheduled Flow Explorer](features/scheduled-flow-explorer.html)

View all active Schedule-Triggered Flows across the org and see when each is next scheduled to run, in List View or Calendar View.

### [Unused Resources](features/unused-resources.html)

Scan the currently open Flow for variables, formulas, text templates, and other resources that are defined but never used.

### [Flow List Search](features/flow-list-search.html)

Search and filter the Salesforce Setup Flows list by label, API name, status, and type.

### [Canvas Search](features/canvas-search.html)

Search the Flow canvas, connector labels, and toolbox items, then move between results quickly.

### [API Name Generator](features/api-name-generator.html)

Generate standardised API names for Flow elements and resources using configurable naming patterns and prefixes.

## More features

- [Flow Version Manager](features/flow-version-manager.html) — bulk-select and delete eligible Flow versions more safely from the versions page.
- [Comparison Exporter](features/comparison-exporter.html) — export Flow version comparison results into an Excel workbook using a structured template.
- [Missing Description Flags](features/missing-description-flags.html) — highlight elements, resources, and Flow-level items that are missing descriptions.
- [Setup Tabs](features/setup-tabs.html) — add quick-access Setup tabs for key Flow and automation destinations.
- [Flow Trigger Explorer Enhancer](features/flow-trigger-explorer-enhancer.html) — enrich Flow Trigger Explorer rows with inline metadata, context tags, and compact tooltips.
- [Flow Metadata & AI Assistant](features/flow-metadata-ai-assistant.html) — work with Flow metadata, copy/download JSON, and generate prompt-ready content for AI-assisted workflows.
