# Getting Started

## Overview

SF Flow Utility Toolkit is a browser extension intended to enhance the Salesforce Flow Builder experience.

## Before you begin

Make sure the extension is installed in your browser and enabled for supported Salesforce domains.

## Supported environments

The extension is intended for use with Salesforce environments such as:

- `*.salesforce.com`
- `*.salesforce-setup.com`
- `*.lightning.force.com`
- `*.my.salesforce.com`
