# Getting Started

## Overview

SF Flow Utility Toolkit is a browser extension designed to enhance Salesforce Flow Builder and related Salesforce automation pages with focused productivity, navigation, documentation, and analysis features.

The toolkit is intended to support Flow builders, administrators, and consultants by adding practical utilities directly into the Salesforce experience.

# Please be aware that if you use the extension may see times where the Chrome version is more up-to-date than the Microsoft Edge version. This is due to Microsoft Edge taking up to 7 business days to review and approve changes.

## Before you begin

Before using the toolkit, make sure that:

- the extension is installed in your browser
- the extension is enabled
- you are working in a supported Salesforce environment
- you have access to the relevant Salesforce pages for the features you want to use

Some features are designed for Flow Builder specifically, while others are intended for related Setup and automation pages.

## Install the extension

Should you need to install the extension, you can do so on the following browsers:

- <a href="https://chromewebstore.google.com/detail/sf-flow-utility-toolkit/mjbmlikmdkcakcbilibhbgcjdnidkpfl" target="_blank" rel="noopener noreferrer">Google Chrome</a>

## Supported environments

The extension is intended for use with Salesforce environments such as:

- `*.salesforce.com`
- `*.salesforce-setup.com`
- `*.lightning.force.com`
- `*.my.salesforce.com`

## What the toolkit includes

The current feature set includes tools for:

- Flow analysis and review
- Flow search and navigation
- API naming support
- version management
- comparison export
- missing-description visibility
- Flow Trigger Explorer enrichment
- AI-assisted documentation and prompt preparation

## Feature locations

Different features appear in different parts of Salesforce.

Examples include:

- **Flow Builder** for features such as Canvas Search, Flow Health Check, API Name Generator, Missing Description Flags, and AI Assistant
- **Setup Flows** for Flow List Search
- **Flow details / versions pages** for Flow Version Manager
- **Compare Versions** for Comparison Exporter
- **Flow Trigger Explorer** for Flow Trigger Explorer Enhancer
- **Setup tab bar** for Setup Tabs
