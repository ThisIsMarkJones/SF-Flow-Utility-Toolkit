SF Flow Utility Toolkit is now available on the Chrome Web Store.

This release includes the initial public feature set, including:

* Flow Health Check
* Flow List Search
* Canvas Search
* API Name Generator
* Flow Version Manager
* Comparison Exporter
* Missing Description Flags
* Setup Tabs
* Flow Trigger Explorer Enhancer
* Flow Metadata & AI Assistant

To download the extension to your Chrome Browser, please visit:

[Google Chrome](https://chromewebstore.google.com/detail/sf-flow-utility-toolkit/mjbmlikmdkcakcbilibhbgcjdnidkpfl)
