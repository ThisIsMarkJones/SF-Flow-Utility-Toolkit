# SF Flow Utility Toolkit

A Chrome / Edge browser extension for Salesforce Flow Builder.

SF Flow Utility Toolkit is designed to make working with Salesforce Flows faster and easier by adding practical utilities directly into the Flow Builder experience.

## Status

This project is currently approaching its **V1 release** for submission to the Chrome Web Store and Microsoft Edge Add-ons store.

## What it does

The extension provides a growing set of tools for Salesforce Flow Builder, with functionality focused on areas such as:

- searching and navigating Flows
- analysing Flow metadata and health
- managing Flow versions
- generating API names and other helper outputs
- supporting comparison and documentation workflows

## Supported environment

SF Flow Utility Toolkit is intended for use with Salesforce environments accessed through:

- `*.salesforce.com`
- `*.salesforce-setup.com`
- `*.lightning.force.com`
- `*.my.salesforce.com`
